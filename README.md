# egress-leyu-lab

## 项目简介

这是一个用于归档和发布独立 HTML 页面的仓库。仓库内包含多个不同用途的静态页面，通常用于快速访问、测试或演示目的。本仓库不针对任何特定域名或网站，仅作为 HTML 文件的集中存放与版本管理场所。

## 目录说明

```
egress-leyu-lab/
├── index.html          # 仓库首页或导航页
├── pages/              # 存放各类独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── style.css
│   └── ...
└── README.md           # 本文件
```

- `pages/`：主要存放归档的独立 HTML 页面，每个文件通常为一个完整页面。
- `assets/`：用于存放页面中引用的公共资源，如样式表、脚本或图片。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部服务或特定后端环境。
- 页面内容可能涉及不同的主题或功能，但均保持静态形式。
- 归档的页面可能不定期更新或清理，以保持仓库内容简洁。

## 维护说明

- 本仓库由维护者根据实际需求添加或移除页面。
- 不保证所有页面长期可用或兼容所有浏览器。
- 欢迎通过 Issue 提交建议或反馈，但请注意本仓库不提供具体网站或服务支持。

## 使用方式

您可以通过 GitHub Pages 或直接下载仓库到本地浏览这些 HTML 页面。具体访问路径请参考仓库内目录结构。
